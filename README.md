# 🎨 BgColorApp

A simple and interactive web application that allows users to change the background color dynamically with a single click.

## 🚀 Live Demo
🔗 [View Live](https://github.com/deepthinkss/BgColorApp)

---

## 📌 About the Project

BgColorApp is a beginner-friendly project designed to demonstrate how JavaScript can be used to manipulate the DOM and dynamically update UI elements.  

With just a click, users can generate and apply random background colors, making it a fun and interactive experience.

---

## ✨ Features

- 🎨 Change background color instantly
- 🔁 Generate random colors on each click
- ⚡ Lightweight and fast
- 📱 Responsive design
- 🧠 Beginner-friendly code structure

---

## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling
- **JavaScript (Vanilla JS)** – Functionality
